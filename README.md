# Credit Risk Scorecard - SQL Analysis

Comprehensive credit risk analysis using SQL on UCI Credit Card Default Dataset (30,000 customers).

## Project Overview

- **Objective**: Analyze default risk factors using SQL Server
- **Dataset**: 30,000 customer records, 25 features
- **Tools**: SQL Server, SSMS
- **Status**: Complete

## Key Findings

| Metric | Value |
|--------|-------|
| **Default Rate** | 22.12% |
| **Total Customers** | 30,000 |
| **Total Defaults** | 6,636 |
| **Strongest Predictor** | Payment Behavior |
